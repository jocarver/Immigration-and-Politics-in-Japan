## 治変数 (treatment variable) 構築への示唆

都道府県 × 在留資格 × 年の集計テーブルを作る場合:

```python
import pandas as pd

df = pd.read_excel("24-12-t1.xlsx", sheet_name="令和6年12月末", engine="openpyxl")

# 都道府県 × 在留資格 集計
agg = (
    df.groupby(["都道府県", "在留資格"])["在留外国人数"]
    .sum()
    .reset_index()
)
```

## 注意事項

1. **技能実習の欠如**: 2024-12時点のデータには「技能実習」が存在せず「育成就労」に移行済み
   - 2022年以前: `28：技能実習`
   - 2024年以降: `42：育成就労` (または廃止・統合された在留資格)
2. **特定技能**: 2019年4月新設。それ以前の期間は列がない (在留資格が増える)
3. **年齢カラムの扱い**: `年齢` 列は個人単位の実年齢。集計時は `年齢（５歳階級）` か全集計を推奨
4. **全国行**: `都道府県` が `00：全国` の行は除外してから都道府県単位集計する
